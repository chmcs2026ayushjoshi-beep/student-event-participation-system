# MERN Participation Management System

## Project Overview

The MERN Participation Management System is a full stack web application built using the MERN stack (MongoDB, Express.js, React.js, Node.js). The system allows users to create, view, update, and delete student participation records in an organized way.

It provides a simple and responsive interface where participation details can be managed efficiently. The system also includes department-based filtering to easily view records according to different departments.

## MERN Stack Used

MongoDB – Database used to store participation data
Express.js – Backend framework used to build REST APIs
React.js – Frontend library used to build the user interface
Node.js – Runtime environment used to run the backend server

## Installation and Setup

### 1. Clone the Repository

git clone https://github.com/yourusername/your-repository-name.git

### 2. Install Backend Dependencies

cd backend
npm install

### 3. Install Frontend Dependencies

cd frontend
npm install

### 4. Configure Environment Variables

Create a `.env` file inside the backend folder.

Example:

MONGO_URI=your_mongodb_connection_string
PORT=5000

### 5. Run the Application

Start Backend Server

npm start

Start Frontend

npm run dev
